Umrah Sheba

Umrah Sheba is a comprehensive web-based platform designed to facilitate the management and booking of religious pilgrimage packages, including Umrah and Hajj. The system incorporates a multi-level user hierarchy including administrators, users, agents, and sub-agents, each with distinct permissions. It enables seamless package creation, customization, and booking, providing a user-friendly experience for managing Umrah and Hajj services. The project includes dashboards for different user roles, an invoice system, and various other features tailored to ensure smooth management of religious trips.

As a Full Stack Developer, I was responsible for designing and implementing both front-end and back-end features. I developed the Umrah and Hajj package creation and booking systems, customized dashboards for various user roles, and integrated an invoice generation system to streamline financial tracking. I ensured the system's usability across different devices with responsive design.

I worked in a team of 3-5 developers using Agile methodology. I contributed to system architecture discussions, assisted with database design, and led key aspects of the role-based access control system. We collaborated through regular stand-ups, ensuring smooth progress and quick resolution of issues.

  • Backend: Laravel, MySQL
  • Frontend: HTML5, CSS3, Bootstrap, JavaScript, jQuery
  • Authentication: Laravel Passport, Role-based access control
  • Version Control: Git
  • UI Libraries: Google Fonts, Font Awesome, Bootstrap Icons, Owl Carousel, Lightbox, Animate.css
  • Role Management System: Admins can assign roles (admin, user, agent, sub-agent) and manage access permissions.
  • Umrah and Hajj Package Booking: Users can browse, book, and customize packages with live availability updates.
  • Custom Package Builder: Allows users to create personalized pilgrimage packages based on preferences like travel dates, accommodation, and transport options.
  • Dashboards: Special dashboards for agents and sub-agents to manage bookings, view invoices, and handle clients.
  • Invoice System: Automated invoice generation for users, agents, and sub-agents to track payments and bookings.
  • Responsive Design: Optimized for desktop and mobile platforms to ensure accessibility from any device.
Challenge:

One of the main challenges faced during the development of Umrah Sheba was managing the complex user permissions and roles. With multiple user levels (administrators, agents, users, and sub-agents), it was crucial to ensure that each role had the appropriate access to specific functionalities without compromising the security and integrity of the platform.

Solution:

To address this challenge, I implemented a robust role-based access control (RBAC) system using Laravel's built-in authorization features. This allowed me to define user roles and permissions clearly, ensuring that each user could only access the features relevant to their role. Additionally, I created an intuitive user interface that provides role-specific dashboards, enhancing usability and helping users navigate the platform efficiently.

The Umrah Sheba platform successfully streamlined the entire process of booking religious pilgrimages, from browsing packages to booking and generating invoices. The role-based system provided security and ensured that the right users had access to the right features, enhancing trust in the system. The intuitive dashboards and automated systems reduced manual workload for agents and administrators, improving operational efficiency. Positive feedback from early adopters indicated high satisfaction with the customizable package options and the seamless user experience.

Challenge:

One of the main challenges was integrating the extensive settings module, which includes multiple sub-sections such as About Us, Mission & Vision, and Executive Director details. Ensuring that each setting was user-friendly and seamlessly integrated with other modules required meticulous planning and execution.

Solution:

To address this, I implemented a modular architecture using Laravel's MVC framework, allowing each setting section to be developed and maintained independently. I utilized Laravel's Blade templating engine to create reusable components, ensuring consistency across different settings. Additionally, I conducted user testing sessions to gather feedback and iteratively improve the user interface, making it more intuitive and efficient for administrators to manage various settings.

The **Unnayan Sangstha** platform was successfully deployed, providing a robust solution for managing the organization's administrative and operational tasks. The system's intuitive interface and comprehensive feature set have significantly enhanced operational efficiency, improved data management, and facilitated better communication within the organization. Feedback from stakeholders has been overwhelmingly positive, highlighting the platform's effectiveness in streamlining workflows and supporting the organization's mission.

Human Development For Poor Society (HDPSBD)

The HDPSBD platform is an integrated management system designed to uplift poor communities by facilitating various programs and services. It includes features for managing user roles, online payments, project tracking, and donation management, ensuring effective service delivery and community engagement.

As a Full Stack Developer for the HDPSBD project, I was responsible for both front-end and back-end development. My tasks included designing user interfaces using Bootstrap, implementing server-side logic with Laravel, and managing databases with MySQL. I also worked on integrating payment systems and role-based access features.

Collaborated with a team of 5 developers using Agile methodologies. My contributions included participating in sprint planning, conducting code reviews, and developing core features like user management, project tracking, and donation processing. Regular stand-up meetings facilitated effective communication and project progress tracking.

  • Backend: Laravel, MySQL
  • Frontend: Bootstrap, HTML5, CSS3, JavaScript, jQuery
  • APIs: RESTful APIs for third-party integrations
  • Version Control: Git
  • Payment Gateway: SSL Commerz
  • User Management: Manage admin and user roles with specific permissions.
  • Online Payment: Secure payment processing through SSL Commerz.
  • Project Tracking: Tools for monitoring ongoing projects and their statuses.
  • Donation Management: Facilitate donations through various channels and manage donor information.
  • Multi-language Support: Bilingual support in Bangla and English for user accessibility.

Challenge: One of the main challenges was ensuring data security and user adaptation to the new system, particularly in managing sensitive donor information and project details.

Solution: We implemented encryption for data security and conducted user training sessions to familiarize stakeholders with the platform, enhancing user engagement and confidence in using the system.

The HDPSBD platform has effectively enhanced operational efficiency and service delivery. With robust user management and payment systems, it supports the organization's mission to uplift poor communities. The responsive design and bilingual support have improved accessibility and engagement with stakeholders.
